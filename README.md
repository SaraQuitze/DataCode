## Project Overview

A complete Data Analytics solution for a call center, demonstrating professional ETL processes, dimensional modeling, and business intelligence reporting. This project showcases skills required for a Data Analyst Junior position.

### Business Problem
The call center needs to track:
- Customer satisfaction (CSAT scores 1-5)
- Agent productivity and resolution rates
- Service Level Agreement (SLA) compliance
- Employee turnover and retention metrics
- Daily operational trends
